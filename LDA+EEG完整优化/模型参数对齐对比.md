# LDA 与 EEGNet 参数对齐对比

## 对齐原则

- 共同参数：时间窗口、通道集合、字符级交叉验证、事件级 BalAcc/AUC、Unknown 正确数。
- LDA 独有参数：`bins`、`shrinkage`、类别先验、`StandardScaler`、字符聚合方式。
- EEGNet 独有参数：`epochs`、`lr`、`dropout`、`pos_weight`、字符聚合方式 `top3`。

说明：两个模型不能强行使用完全相同的内部参数。LDA 的 `StandardScaler` 和 `trimmed_mean` 是线性模型进一步优化的一部分；EEGNet 保持 `top3` 字符聚合，不直接套用 LDA 的 `trimmed_mean`。

## 结论

在统一 `0-500 ms + LDA Top-12` 的主要口径下，LDA 进一步加入 `StandardScaler` 和 `trimmed_mean` 后达到 CV `11/12`、Unknown `7/8`；EEGNet 使用 `top3` 聚合达到 CV `9/12`、Unknown `7/8`。因此报告中应区分“基础对齐配置”和“模型独有最优配置”：对齐配置用于公平比较窗口和通道，最优配置用于展示各模型最终性能。
