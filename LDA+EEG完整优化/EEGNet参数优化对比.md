# EEGNet 参数优化对比报告

## 说明

本报告记录 EEGNet 模型的参数优化过程。为了和 LDA 对齐，EEGNet 使用同一组时间窗口和通道方案：`0-600`、`0-500`、`0-400`、`100-500`、`150-450`，并分别补跑全 20 通道和 LDA Top-12 通道两种配置。

## 优化目标

EEGNet 的优化目标与 LDA 保持一致，重点比较三类结果：

- 字符级交叉验证正确数：主要指标，反映模型能否稳定识别训练字符。
- 事件级 Balanced Accuracy 和 AUC：辅助指标，反映每次闪烁事件的分类能力。
- Unknown 字符正确数：最终参考指标，用来观察模型对未知字符的泛化效果。

## 共同对齐参数

EEGNet 与 LDA 对齐的共同实验参数如下：

| 参数 | 取值 | 说明 |
| --- | --- | --- |
| 时间窗口 | `0-600`、`0-500`、`0-400`、`100-500`、`150-450` | 与 LDA 使用同一组窗口 |
| 通道方案 | 全 20 通道、LDA Top-12 通道 | 比较全通道输入和通道筛选后的输入 |
| 主要评价 | CV 字符正确数 | 与 LDA 使用同一评价口径 |
| 辅助评价 | 事件 BalAcc、事件 AUC、Unknown 正确数 | 用于解释模型稳定性和泛化表现 |

## 模型独有参数

EEGNet 没有 LDA 的 `bins`、`shrinkage` 和显式线性判别参数。它的独有参数主要来自神经网络训练过程和字符聚合方式：

| 参数 | 当前取值 | 含义 |
| --- | --- | --- |
| `epochs` | 80 | 每个折次的训练轮数 |
| `lr` | 0.001 | 学习率 |
| `dropout` | 0.5 | Dropout 正则化强度 |
| `pos_weight` | 1.0 | 正类损失权重 |
| 字符聚合 | top3 | 对同一行/列多次闪烁得分取最高 3 次平均，再选字符 |

## 优化过程与结果

第一步是固定 EEGNet 独有训练参数。为了让结果主要反映窗口和通道的影响，本轮实验统一使用 `epochs=80`、`lr=0.001`、`dropout=0.5`、`pos_weight=1.0`，字符聚合统一使用 `top3`。这样 EEGNet 的表格就能和 LDA 的窗口、通道实验直接对比。

第二步是确定字符聚合方式。EEGNet 对每次闪烁事件输出目标概率，一个字符需要把对应行和列的多次闪烁分数聚合后再选择最大行列。`top3` 的含义是只取同一行/列得分最高的 3 次求平均，比直接 mean 更不容易被偶发低分拖低。前期结果显示 top3 对 Unknown 字符更稳定，因此本轮所有 EEGNet 对齐实验固定使用 top3。

第三步是窗口对比。全 20 通道下，`0-400 ms` 的 CV 字符正确数最高，为 `9/12`，事件 AUC `0.809`，Unknown `6/8`；`0-500 ms` 为 `8/12`，事件 AUC `0.803`，Unknown `5/8`；`0-600 ms` 为 `8/12`，事件 AUC `0.800`，Unknown `4/8`。这说明 EEGNet 在全通道输入下对较短窗口更敏感，过长窗口可能引入后段噪声。`100-500 ms` 和 `150-450 ms` 分别只有 `6/12` 和 `5/12`，说明去掉早期响应后字符识别下降。

第四步是通道对比。使用 LDA Top-12 通道后，多数窗口的事件 AUC 提高。最关键的是 `0-500 ms`：全 20 通道为 CV `8/12`、AUC `0.803`、Unknown `5/8`，切换到 Top-12 后提升为 CV `9/12`、AUC `0.834`、Unknown `7/8`。`0-600 ms + Top-12` 的 AUC 最高，为 `0.840`，但 CV 字符正确数只有 `7/12`；`0-400 ms + Top-12` 的 Unknown 也达到 `7/8`，但 CV 只有 `7/12`。因此综合看，`0-500 ms + LDA Top-12 + top3` 是更均衡的 EEGNet 配置。

## 通道参数对比

| 重点配置 | 全 20 通道结果 | LDA Top-12 结果 | 对比结论 |
| --- | --- | --- | --- |
| 0-600 | CV `8/12`，AUC `0.800`，Unknown `4/8` | CV `7/12`，AUC `0.840`，Unknown `6/8` | Top-12 提升 AUC 和 Unknown，但 CV 下降 |
| 0-500 | CV `8/12`，AUC `0.803`，Unknown `5/8` | CV `9/12`，AUC `0.834`，Unknown `7/8` | Top-12 三类指标都更好，是主推荐配置 |
| 0-400 | CV `9/12`，AUC `0.809`，Unknown `6/8` | CV `7/12`，AUC `0.828`，Unknown `7/8` | Top-12 提升 AUC/Unknown，但 CV 下降 |
| 100-500 | CV `6/12`，AUC `0.783`，Unknown `6/8` | CV `7/12`，AUC `0.804`，Unknown `6/8` | Top-12 略有提升，但整体不如 0-500 |
| 150-450 | CV `5/12`，AUC `0.782`，Unknown `6/8` | CV `6/12`，AUC `0.786`，Unknown `5/8` | 整体表现较弱 |

## 推荐配置

EEGNet 推荐配置为：`0-500 ms + LDA Top-12 通道 + top3 字符聚合 + epochs=80 + lr=0.001 + dropout=0.5 + pos_weight=1.0`。

该配置结果为 CV 字符正确 `9/12`，事件 BalAcc `0.762`，事件 AUC `0.834`，Unknown `7/8`。它不是单项 AUC 最高的配置，但在字符交叉验证、事件排序和 Unknown 泛化之间最均衡。

## 结论

EEGNet 的补跑结果说明，通道筛选对神经网络同样有效。尤其在 `0-500 ms` 窗口下，LDA Top-12 通道同时提升了 CV 字符正确数、事件 AUC 和 Unknown 正确数。与 LDA 一致，`0-500 ms` 可以作为两类模型共同对比的主窗口，Top-12 通道可以作为统一的优化通道方案。
