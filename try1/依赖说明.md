# P300 Speller Notebook 依赖说明

本文档对应 `P300_Speller_Experiment.ipynb`，用于说明运行该 notebook 所需的 Python 环境和第三方库。

## 1. 推荐环境

推荐使用 Python 3.10 或 Python 3.11。

建议在独立虚拟环境中运行：

```bash
python -m venv .venv
.venv\Scripts\activate
```

如果使用 Anaconda，也可以创建 conda 环境：

```bash
conda create -n p300 python=3.11
conda activate p300
```

## 2. 必需依赖

Notebook 使用到以下第三方库：

| 依赖 | 用途 |
|---|---|
| numpy | 数值计算、矩阵处理 |
| pandas | 读取 Excel、整理事件表和结果表 |
| openpyxl | pandas 读取 `.xlsx` 文件所需引擎 |
| scipy | EEG 滤波、陷波、信号处理 |
| scikit-learn | LDA、Logistic Regression、交叉验证与评价指标 |
| matplotlib | ERP、事件序列、混淆矩阵、行列得分绘图 |
| ipython | Jupyter 中的表格展示 |
| jupyter | 打开和运行 notebook |

## 3. pip 安装命令

```bash
pip install numpy pandas openpyxl scipy scikit-learn matplotlib ipython jupyter
```

如果希望固定较稳定的版本，可使用：

```bash
pip install ^
  numpy>=1.24 ^
  pandas>=2.0 ^
  openpyxl>=3.1 ^
  scipy>=1.10 ^
  scikit-learn>=1.3 ^
  matplotlib>=3.7 ^
  ipython>=8.0 ^
  jupyter>=1.0
```

PowerShell 中也可以写成一行：

```powershell
pip install "numpy>=1.24" "pandas>=2.0" "openpyxl>=3.1" "scipy>=1.10" "scikit-learn>=1.3" "matplotlib>=3.7" "ipython>=8.0" "jupyter>=1.0"
```

## 4. conda 安装命令

```bash
conda install numpy pandas openpyxl scipy scikit-learn matplotlib ipython jupyter
```

或使用 conda-forge：

```bash
conda install -c conda-forge numpy pandas openpyxl scipy scikit-learn matplotlib ipython jupyter
```

## 5. 可选依赖

当前 notebook 没有强制使用以下库，但如果后续扩展 EEG 分析，可以考虑安装：

| 依赖 | 用途 |
|---|---|
| mne | 更专业的 EEG/ERP 数据结构、滤波、epoch、可视化 |
| seaborn | 更美观的统计图 |
| pyriemann | 协方差和黎曼几何 EEG 分类方法 |

安装命令：

```bash
pip install mne seaborn pyriemann
```

## 6. 文件和路径要求

运行 notebook 时，建议保持如下目录结构：

```text
sy2/
├─ P300_Speller_Experiment.ipynb
├─ 实验说明.txt
├─ 依赖说明.md
└─ P300-S1/
   ├─ S1_train_data.xlsx
   ├─ S1_train_event.xlsx
   ├─ S1_test_data.xlsx
   └─ S1_test_event.xlsx
```

Notebook 默认使用：

```python
PROJECT_DIR = Path.cwd()
DATA_DIR = PROJECT_DIR / "P300-S1"
```

因此，打开 notebook 后，当前工作目录应为 `sy2`。如果不是，可以手动修改 notebook 中的 `PROJECT_DIR` 或 `DATA_DIR`。

## 7. 常见问题

### 读取 Excel 报错

如果出现 `Missing optional dependency 'openpyxl'`，说明缺少 Excel 读取引擎：

```bash
pip install openpyxl
```

### 中文字体显示为方块

Notebook 中已设置常见中文字体：

```python
plt.rcParams["font.sans-serif"] = ["SimHei", "Microsoft YaHei", "Arial Unicode MS", "DejaVu Sans"]
```

如果仍然显示异常，可安装或切换系统已有中文字体。

### scipy 或 scikit-learn 安装较慢

Windows 环境下建议优先使用 conda：

```bash
conda install scipy scikit-learn
```

### Jupyter 无法启动

确认安装了 jupyter：

```bash
pip install jupyter
jupyter notebook
```

或使用 VS Code / PyCharm 直接打开 `.ipynb` 文件运行。
